# 生成AI 中級編 ゼミ

## 📚 概要

本ゼミは、生成AIやITに一定の基礎知識を持つ受講生を対象に、生成AIをより高度に、実践的に使いこなす力を身につけることを目的とした、実践的なハンズオン型の学習プログラムです。

### 🎯 目的

単に「AIを使ってみる」段階から一歩進み、生成AIをどのように組み合わせ、設定し、目的に応じて活用すれば価値を生み出せるのかを、実際に手を動かしながら学びます。

生成AI関連のツールや仕組みを題材に、AIがどのように動き、どこに強みや限界があるのかを理解したうえで、現実の課題にどう応用できるかを考えます。講義だけでなくハンズオン形式の演習を通じて、「知識として知っているAI」を「自分で設計し、使いこなせるAI」へと変えていくことを重視します。

### 👥 対象者

* 生成AIの基礎知識を持っている方（ChatGPTなどを使用した経験がある）
* プログラミングの基礎知識がある方（任意、一部のセクションで必要）
* 生成AIを実務やプロジェクトで活用したいと考えている方
* AIツールの仕組みを深く理解し、応用力を身につけたい方

### 🎓 学習到達目標

本ゼミを修了することで、以下のスキルを習得できます：

1. **生成AIツールの設計・構築能力**
   - ノーコードツール（Dify）を使った生成AIアプリケーションの開発
   - プロンプトエンジニアリングの実践的なテクニック
   - RAG（Retrieval-Augmented Generation）の実装と活用

2. **先進的なAI連携技術の理解**
   - Model Context Protocol（MCP）の概念と活用方法
   - AIエージェントの設計と外部システムとの連携
   - API統合による高度な自動化の実現

3. **実践的な問題解決能力**
   - 業務課題に対する生成AIソリューションの設計
   - AIの制約や限界を理解した上での最適な活用方法の選択
   - 運用・保守を考慮したAIシステムの構築

## 📖 カリキュラム構成

本ゼミは以下の2つの主要テーマで構成されています：

### 1. Dify基礎（全5回・90分/回）

**Dify**は、プログラミング不要でAIアプリケーションを構築できるノーコードプラットフォームです。ビジュアルな操作で、RAGや外部API連携を含む高度なAIアプリケーションを開発できます。

| 回次 | テーマ | 主な学習内容 |
|------|--------|--------------|
| 第1回 | Dify導入と環境構築 | アカウント作成、モデルプロバイダー設定（Gemini）、基本操作 |
| 第2回 | LLMノードとプロンプト設計 | プロンプトエンジニアリング、モデルパラメータ調整、条件分岐 |
| 第3回 | Knowledge（RAG）導入 | ドキュメントアップロード、ナレッジベース構築、検索精度向上 |
| 第4回 | エージェントと外部連携 | HTTPノード、外部API連携、ワークフロー設計 |
| 第5回 | デプロイと発表 | 公開設定、運用、成果発表 |

**最終成果物例:**
- 社内FAQチャットボット（RAG活用）
- Google Sheets要約アシスタント（API連携）
- タスク自動化アシスタント

### 2. Model Context Protocol（MCP）基礎

**MCP**は、AIアシスタントが外部ツールやデータソースと安全に連携するためのオープンスタンダードです。Claude、ChatGPT、その他のAIツールが、ファイルシステム、データベース、APIなどと統合的に機能する仕組みを学びます。

## 🛠️ 使用するツール・技術

### 必須ツール

* **Dify**: AIアプリケーション開発プラットフォーム
  - Webブラウザ（Chrome/Edge推奨）
  - Difyアカウント（無料プランで開始可能）

* **Google AI Studio**: LLMモデルプロバイダー
  - Google Gemini API（無料枠あり）

### 推奨環境

* **Node.js**: v18.0.0以上（LTS版推奨）
* **パッケージマネージャー**: pnpm
* **エディタ**: Visual Studio Code（推奨）

## 🚀 セットアップ方法

### 1. リポジトリのクローン

```bash
git clone <repository-url>
cd generative-ai-intermediate
```

### 2. 依存関係のインストール

```bash
# pnpmを使用
pnpm install

# または npm
npm install
```

### 3. 各セクションの学習開始

各ディレクトリの `README.md` に従って学習を進めてください：

* Difyを学習する場合: [dify-basics/README.md](dify-basics/README.md)
* MCPを学習する場合: [mcp-basics/README.md](mcp-basics/README.md)
